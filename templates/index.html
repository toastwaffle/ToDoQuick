Welcome to ToDoQuick

ToDoQuick is a ToDo system based on the concepts of digital literacies from Mark Hurst's book Bit Literacy. It uses several concepts you may not find with other systems:

  • Delayed task activation
  • Infinitely nestable tasks
  • Infinitely nestable tags
  • Descriptive Projects
  • Daily reminder emails

ToDoQuick is still under active development. It is a one-man project, not yet ready for the big time. There are some features we're looking to implement soon:

  • Shared projects
  • Add task by email
  • Mobile Version
